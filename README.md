## Start
php > 8.1
- composer install
- cp .env.example .env
- настройка данных бд (DB_DATABASE_TEST - тестовая база)
- php artisan key:generate
- php artisan migrate 
- php artisan db:seed 
- php artisan passport:install

Администратор user с id = 1, пароль - password
